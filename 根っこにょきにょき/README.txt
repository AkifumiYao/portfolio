【ゲーム説明】
根っこをひたすら伸ばすスコアアタックゲームです。
左上の水分ゲージが無くなると、成長が止まりプレイ終了です。
水分は根っこの成長過程で地中に存在する水だまりに触れると回復できます。

【コンセプト】
基本は運だが細かいテクニックは実力を出せるゲームを考えました。
しかし、現状は進む方向に水だまりが存在するかしないかの運任せになってしまっているので
拡張アイデアとして
・近くの水だまりがある方向、もしくは近くを通ったら音がする。
・水の近くには苔を生やし、水が近いことを知らせる。
・地層が変わるごとに根っこを分岐させられる権利を獲得し、幅広く探索が出来る。
これらを考えています。
