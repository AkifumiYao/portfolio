【ゲーム説明】
画面中央上部から飛んでくる寿司を指定場所でタイミングよくタップする
スマホ向けリズムゲームです。
お客さんに寿司が当たるとお客さんがダメージを受け、お客さんの誰か一人でも体力が
ゼロになればプレイ終了です。
特徴は、寿司が画像ではなく３Dのモデルということです。
３Dにすることで実際に飛んでくる感を出したかったのですが、現状画像でも変わらない
仕様になってしまいました。

【コンセプト】
初めは回転寿司をイメージし、レーンで回ってくる寿司をタイミングよくタップする
リズムゲームを考えましたが、仕様を考えているうちに「太鼓の達人」シリーズと
似てしまったと思い、寿司自体を回転させることにしました。
その結果、また別のゲームに似てしまいました。。。
